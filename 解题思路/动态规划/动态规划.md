## 动态规划套路

**明确 base case -> 明确「状态」-> 明确「选择」 -> 定义 dp 数组/函数的含义**。

```python
# 初始化 base case
dp[0][0][...] = base
# 进行状态转移
for 状态1 in 状态1的所有取值：
​    for 状态2 in 状态2的所有取值：
​        for ...
​            dp[状态1][状态2][...] = 求最值(选择1，选择2...)
```
